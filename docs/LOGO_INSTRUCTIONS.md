# Instructions pour le Logo ONATECH

## Emplacement du logo

Le logo doit être placé dans le dossier `public/` avec le nom exact :
```
public/logo-onatech.png
```

## Spécifications recommandées

- **Format** : PNG avec fond transparent
- **Taille minimale** : 200x200px
- **Taille recommandée** : 400x400px pour une meilleure qualité
- **Couleur** : Le logo original ONATECH (bleu)
- **Fond** : Transparent (PNG avec alpha channel)

## Comment ajouter le logo

1. **Téléchargez votre logo ONATECH** au format PNG
2. **Renommez-le** en `logo-onatech.png`
3. **Placez-le** dans le dossier `public/` à la racine du projet
4. **Vérifiez** que le logo s'affiche correctement dans le header

## Vérification

Après avoir ajouté le logo, vérifiez que :
- Le logo s'affiche dans le header (en haut à gauche)
- Le logo est bien dimensionné (environ 48x48px dans le header)
- Le logo a un fond transparent
- Le logo est net et de bonne qualité

## Fallback

Si le logo n'est pas trouvé, un icône de fallback (bouclier bleu) s'affichera automatiquement.

## Utilisation dans l'application

Le logo est utilisé dans :
- **Header** : Affiché à côté du nom "ONATECH" en haut de chaque page
- **Favicon** : Peut être utilisé comme favicon (optionnel)
- **Métadonnées** : Peut être référencé dans les métadonnées Open Graph (optionnel)
