# Guide pour générer les favicons

Pour générer les favicons à partir du logo ONATECH, vous pouvez utiliser un outil en ligne comme :

1. **Favicon.io** : https://favicon.io/favicon-converter/
2. **RealFaviconGenerator** : https://realfavicongenerator.net/
3. **Favicon Generator** : https://www.favicon-generator.org/

## Étapes :

1. Téléchargez le logo `/public/logo-onatech.png`
2. Allez sur un des sites ci-dessus
3. Uploadez le logo
4. Générez les favicons dans les tailles suivantes :
   - `favicon.ico` (16x16, 32x32, 48x48)
   - `icon-192.png` (192x192)
   - `icon-512.png` (512x512)
   - `apple-icon.png` (180x180)

5. Placez tous les fichiers générés dans le dossier `/public/`

## Alternative : Utiliser le logo directement

Si vous préférez utiliser le logo existant temporairement, vous pouvez créer des liens symboliques ou copier le logo avec les noms requis.
